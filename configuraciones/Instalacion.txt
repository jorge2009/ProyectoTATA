El proyecto esta realizado con Spring boot
Lenguaje JAVA
JDK 20
Se genera un archivo war.
con el que se publica el API
sobre cualquier Servidor WEB soportado por JAVA
La pruebas esta realizadas sobre Tomcat 10
se debe carga el war en el servidor y consumir el API
En los documentos adjunto se encuentran las pruebas realizadas
para la base de datos se utilizo MySQL
se encuentra adjunto el script de la base de datos
es importante cambiar las credenciales dentro del proyectos en properties para tener acceso a la base de datos
el contenedor esta generado con las características del archivo de creación y publicación
En aplicacion .properties cambiar la configurqacion de base de datos y usuario
